NB: l'esecuzione degli esempi, per simulare una comunicazione cross-origin, richiede la presenza di due domini distinti.

Gli esempi usano i due host seguenti:

- http://localhost
- http://dominiotest.dev

Il file "index.html" deve trovarsi in "http://localhost".
Il file "child.html" oppure il file "iframe.html" devono trovarsi in "http://dominiotest.dev".
