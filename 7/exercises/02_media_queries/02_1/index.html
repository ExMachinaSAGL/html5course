RESIZE ME
  • Specificare stili diversi a seconda di condizioni specifiche del browser (dimensioni, orientamento...)
  • media type (all, screen [default], print, tv, braille, 3d-glasses...)
  • breakpoint (min-width, max-width, orientation, device-aspect-ratio...)
  • introdurre un breakpoint SOLO quando serve, non a larghezze predefinite
<link href="styles.css" media="all and (max-width: 1024px)">
@import url(styles.css) all and (max-width: 1024px) {...}
@media all and (max-width: 1024px) {...}
  • CSS media queries: developer.mozilla.org/en-US/docs/Web/Guide/CSS/Media_queries
  • W3C Media Queries: www.w3.org/TR/css3-mediaqueries
  • CSS Media Queries & Using Available Space: css-tricks.com/css-media-queries
